## 📱 功能一览

| 功能 | 说明 |
|------|------|
| 🕐 **实时时钟** | 显示当前时、分，自动跟随手机时间 |
| 📅 **四柱八字** | 年柱、月柱、日柱、时柱实时更新 |
| 🌙 **当前时辰** | 显示当令地支（子丑寅卯...）、脏腑、经络 |
| 🔄 **五行属性** | 当前时辰的五行归属（木火土金水） |
| 🏥 **养生提示** | 每个时辰对应的中医养生建议 |
| 📋 **十二时辰总表** | 一目了然查看全天时辰排布 |
| 📲 **可安装到桌面** | PWA 支持，像原生 App 一样使用 |

## 🎨 设计风格

- **墨水屏美学** — 米白底色、古风宋体、低对比度，仿若纸质书卷
- **极简布局** — 无冗余装饰，信息清晰呈现
- **节能友好** — 每 10 秒刷新一次，减少电量消耗
- **离线可用** — 首次加载后，即使在无网络环境也能正常使用

## 🚀 快速使用

### 在线访问

👉 **[点此打开](https://upcloud_blue.gitee.io/ziwuliuzhu-app)**（部署后替换实际地址）

### 安装到手机桌面（推荐）

**📱 华为手机：**
1. 用手机自带**浏览器**（或 Chrome）打开上方链接
2. 页面加载后，点击浏览器底部或顶部的 **「⋮」**（菜单按钮）
3. 在弹出的菜单中选择 **「添加到桌面」**（或「添加到主屏幕」）
4. 预览界面出现，点击右下角 **「添加」** 或 **「确定」**
5. 手机桌面出现「子午流注」图标 ✅

> 以后直接点击桌面图标就能打开，全屏运行，和原生 App 一样流畅！

**其他 Android 手机（小米/三星/OPPO/vivo等）：**
1. 用 Chrome 浏览器打开链接
2. 点击右上角 **「⋮」** → **「添加到主屏幕」**
3. 点击 **「添加」**，桌面出现图标

**iPhone/iPad：**
1. 用 Safari 打开链接
2. 点击底部菜单 **「分享」** 按钮 → **「添加到主屏幕」**
3. 点击右上角 **「添加」**，桌面出现图标

> 💡 **首次打开后，页面会自动缓存**，以后即使没有网络也能正常使用！

### 在电脑上使用

直接在浏览器打开即可，支持所有现代浏览器。
